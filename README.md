# Daily Habit Tracker

## Overview
The Daily Habit Tracker is a web application built using the MERN stack that allows users to add, track, and manage their daily or weekly habits. It provides authentication, progress tracking, and visual streak monitoring to help users build consistency.

## Features
- User registration and login using JWT authentication  
- Add, update, and delete habits  
- Track daily and weekly progress  
- View current streaks and completion status  
- Calendar view for visual progress tracking  

## Tech Stack
- **Frontend:** React.js, Axios, React Router DOM  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB with Mongoose  
- **Authentication:** JWT (JSON Web Token), bcrypt  

## Future Improvements
- Email or push notifications for habit reminders  
- Analytics and performance insights  
- Improved calendar visualization
